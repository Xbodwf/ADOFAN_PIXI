# ADOFAN_PIXI
A Dance Of Pixi and An. ADOFAI Player written by AnStartist using javaScript and PIXI Extension. You Can run it on HTTP-Server.

To rewrite my app , you MUST run 'npm install' in your download directory.

Please Run CMD or Terminal.

Input 'npx serve' or 'http-server' in your download directory.

Or use Python to hold a server , input 'python -m http.server 8000' in your download directory.
